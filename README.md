# 使用说明

- 文章原稿位于`source/_posts` 目录下

- 更新`jasonzeng.top`
  - 文章编辑完成后执行`npm run update`
  - 或者 `./bin/update.sh`
  - 执行后会更新`coding.net`源码,并且登陆阿里云服务器拉取`html`文件,从而更新站点
